# Scope_demo.c

Objectif: comprendre comment fonctionne la portée de variables en C (variables globales, variables locales, variables dans des blocs {} en utilisant printf pour verifier leur accessibilité.

## Instructions

### Créer un programme en C qui: 

---

1. Déclare un variable dans la fonction main
2. Crée un bloc supplémentaire 
3. Déclare une nouvelle variable dans le bloc
4. Affiche les valeurs des deux variables avec printf pour vérifier leur accessibilité.
5. Utilise una variable en dehors de son scope pour vérifier leur accessibilité.

## ⭐Compétences visées

---

- printf pour afficher des valeurs
- variable locale à main
- variable locale à un bloc
- qu’une variable n’existe plus en dehors de son scope
- lire un message d’erreur du compilateur
- une structure C propre (main, return, indentation)
